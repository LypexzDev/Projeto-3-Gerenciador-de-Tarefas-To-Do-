3️⃣ Gerenciador de Tarefas (To-Do)
Descrição

Um sistema para criar, listar, concluir e remover tarefas, com status de concluída ou pendente, armazenando dados em JSON.

Arquivo

tarefas.json – lista de tarefas com id, descricao, concluida (boolean)

Funcionalidades principais

Adicionar tarefa: insere descrição, status inicia como False

Listar tarefas: mostra todas as tarefas com status (✅ ou ❌)

Concluir tarefa: marca a tarefa como concluída (True)

Remover tarefa: remove tarefa pelo ID

Lógica de programação

Booleanos para representar status (concluida)

Loops e condicionais para gerenciar tarefas

Funções bem separadas para cada operação

Persistência em JSON para salvar e carregar tarefas
